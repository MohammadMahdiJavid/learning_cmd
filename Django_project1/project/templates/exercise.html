1. How to change the directory (folder) in Command Prompt (lesson1):




If you want to know what is your current working directory now you should (type : cd.) and it will show your current directory



If you want to go one folder or directory backward like when you press flashback button you should (type: cd.. )



If you want to go to another folder you can use (cd) command for example i want to go to test folder in drive (C:) Then I should type: (cd C:\test) or I can use this command (cd \test)



If you want to go to another drive you should write the drive name wwith a colon then you should press enter for exaple i want to go to (D) drive I should type: (D:)



If you want to go back 3 directories or too many directories back you can use this command type:(cd../../..) or you can use this:(cd..\..\..)



If you want to go to another folder in another drive you can use this command (for example i want to go to test folder in D drive) you should type: (cd /d D:\test)
