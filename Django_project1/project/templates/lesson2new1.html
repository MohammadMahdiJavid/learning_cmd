2. How to rename a file with Command Prompt (lesson2):




Syntax of rename command: type:(rename file_path new_name)



for example i want to change test.txt to test.bat and test.txt is in C:\test directory so im gonna type:(rename C:\test\test.txt test.bat)



if my current directory is (C:\test) I can type: (rename test.txt test.bat)



You can abbreviate rename to : (ren) so instead of writing the second line we can type:(ren C:\test\test.txt test.bat)



if you want to rename a directory its a little different for example i want to change (test) directory and test directory is in this address (C:\test) so I should type:(ren C:\test ali) so my directory name will change to ali



if your current directory is this (C:\) you can type this command:(ren test ali)
